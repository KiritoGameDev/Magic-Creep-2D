# 🧙‍♂️ Magic Creep 2D - DragonSlayers Studios ⚔️


**Magic Creep 2D** é um jogo de sobrevivência arcade frenético, onde você controla uma poderosa Maga em uma masmorra infestada de Succubus. O objetivo é simples: sobreviver o maior tempo possível e bater seu recorde!

Este projeto marca o primeiro lançamento oficial da **DragonSlayers Studios**, focado em entregar uma experiência fluida de **1080p a 60 FPS constantes**.

## 🚀 Destaques Técnicos
* **Alta Performance:** Otimizado para hardware moderno (testado em RTX 4060/Ryzen 5 3600) para garantir gameplay sem quedas de frames.
* **Workflow com IA:** Utilização de ferramentas de Inteligência Artificial para a criação de assets visuais e prototipagem rápida.
* **Interface Dinâmica:** HUD completa com contagem de pontos em tempo real e sistema de recordes (High Score).
* **Lógica Robusta:** Implementação de sinais, timers e máquinas de estado simples em GDScript.

## 🕹️ Como Jogar
1. Baixe o executável na aba de [Releases](https://github.com/SEU-USUARIO/Magic-Creep-2D/releases) ou jogue pelo Itch.io.
2. Use **WASD** ou as **Setas** do teclado para se movimentar.
3. Desvie das Succubus que surgem de todas as direções!

## 🛠️ Ferramentas Utilizadas
* **Engine:** Godot 4.x
* **Linguagem:** GDScript
* **Arte:** IA Generativa + Ajustes manuais
* **Controle de Versão:** Git/GitHub

## 📁 Estrutura do Projeto
* `/assets`: Sprites, fontes e recursos visuais.
* `/scenes`: Cenas do jogo (Menu, Fase, Player).
* `/scripts`: Lógica de programação em GDScript.
* `/songs`: Trilhas sonoras e efeitos de áudio.
